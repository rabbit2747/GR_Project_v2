## 핵심 개념 요약

### GR Core 3요소

| 구성요소 | 역할 |
|----------|------|
| **GR DB** | 보안 지식 저장 (원자화된 데이터) |
| **GR Framework** | 3D 좌표계 기반 분류 체계 |
| **GR Atlas** | 시각화 및 탐색 인터페이스 |

### 온톨로지 기반 아키텍처

```
                    GR Ontology
                        │
        ┌───────────────┼───────────────┐
        │               │               │
    [원자 Atom]    [관계 Relation]  [분류 Taxonomy]
        │               │               │
   지식의 최소단위   원자 간 연결    Layer/Zone/Tag
```

### AI 활용 3가지 방식

1. **RAG**: 질문 → 원자 검색 → LLM 컨텍스트 제공
2. **Fine-tuning**: 온톨로지 → 학습 데이터 → 전문 모델
3. **Agent**: LLM + Graph API → 자율 추론
